# Airbnb Data Analysis Project

## Overview
This project focuses on cleaning and analyzing Airbnb dataset.

## Data Cleaning
- Removed invalid values (negative minimum nights)
- Fixed typos (brookln → Brooklyn)
- Standardized column names

## Analysis
- Price distribution
- Price by room type
- Price by neighborhood

## Key Insights
- Manhattan has higher prices
- Entire homes are more expensive
- Listings with more reviews are more popular
